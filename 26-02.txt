Static web pages display the same content to all users, while dynamic web pages can change based on the user or their actions. 
Dynamic: Generated by a web application, and can interact with databases


web server vs application server

Web servers receive requests from clients and pass them on to application servers. 
Application servers use business logic to generate dynamic content. 
Web servers can act as gateways to balance load and maintain performance. 
Application servers can provide features like transaction management, security, and scalability.
